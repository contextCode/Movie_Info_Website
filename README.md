## Getting Started

### 1. Clone the repository

### 2. Install dependencies:

```shell
npm install
```

### Setup .env file

```bash
PORT=5000
MONGO_URI=Add_Your_MONGO_URI_Here
NODE_ENV=development
JWT_SECRET=Add_Your_JWT_SECRET_Here
TMDB_API_KEY=Add_Your_TMDB_API_KEY_Here
```

### Start the App:

```shell
npm run dev
```
